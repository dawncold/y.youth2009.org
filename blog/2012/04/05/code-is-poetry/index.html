好听的诗

Apr 5th, 2012

今天看到一篇code is poetry的文章，里面提到了一首美妙的诗，读后感觉甚好，抄下来了：

《断章》
你站在桥上看风景，
看风景的人在楼上看你。
明月装饰了你的窗子，
你装饰了别人的梦。
喜欢的就抄走吧。
