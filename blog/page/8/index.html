十年初中

Jan 20th, 2012

十年，一个不长不短的时间，能让一代不成熟的人成熟，能让成熟的人更加成熟：）

现在来看，初中正好卡在了十年的个时间点上，站在这里，看到的是现在的那帮已经成熟的人和当年年轻的我们自己。

今天的每个我们都独具个性，朝着自己喜欢的方向发展着，有的更加动人，有的更加帅气，有的依然沉默，有的变得有些开放……每个人的特点似乎都很容易看出来，不像以前的时候，穿着校服，每个人都似乎一样。

饭吃了好久，酒也喝了好多，最终调戏班长和学习委员，非得让他们成一对鸳鸯不可，在我们外人看来他们好合适，就没有一点别扭之处，可惜殊不知这种事情真不是外人能够左右的了，这个时代的人们都具备独自选择的能力和思想，早已不是那个媒妁之言的时代。不过外人的力量也是不可忽视的，至少他们在我们大家的逼迫下抱了抱：）

曲终人散，我回到家都已经下午4点了，这顿饭着实吃了好久，但和10年相比，这才哪跟哪呀！

ただいま

Jan 17th, 2012

看到Y帆的QQ签名已经改成了ただいま，感觉这样十分亲切，于是我决定写一篇以ただいま为题的日志来纪念这趟旅程。

其实这趟旅程和平时的回家没有什么太大的区别，当然过年的回家会和平日里有点不同——人很多，候车室里塞满了等待回家的灵魂，我也算是其中一个。本来打算买8：10的车票，但很可惜没抢到，只能坐12：20的，宁宁就更可惜了，只能坐13：20的，但不知道为何，我坐得这趟车没有坐满人，就让宁宁他们期前上来了，我们又在一趟车上了。

这趟车更多的是给我们一个梳理自己这一年思绪的机会，虽然我在车上除了吃零食就是听歌睡觉，间或看过几页≪程序员≫、≪爱你≫、百度地图、QQ、微博，别的确实没想，哦，好像还想过枫。上车后给她发了短信，虽然每次都是我主动发，不过已经不会在乎这些事情了，有人可能感觉自己和对方要付出同样的感情才行，但实在没有必要一下子要求对方作出那么多，如果对方是爱你的，倒了一定程度会是那样的，可能还会让你感到厌烦：）

回到家后发现家里的布局发生了比较大的改变，这个布局挺舒服的，至少现在是给我这样的感觉：）

脑残的考试终于结束了

Jan 15th, 2012

是在是有些累了，刚刚吃完考试结束后的庆祝午餐——几个同学团购了N张自助火锅的餐券海吃了一顿，发现自己根本就不可能吃回来团购花的钱，而且还是远远不够。我大约吃了一盘子牛羊肉，吃了不少水果、各种蘑菇、海鲜，还算是值了的那种：）

再说这考试吧，几乎是全国最晚放假的一批吧？成功击败了全国99%的学校！回到家正好是小年的晚上，直接下车去爷爷家吃水饺。这就不错了，宁宁订的车比我晚一个小时，回到家还不得六七点啊，但那时候也不晚，刚刚开吃：）

考试前复习得好累，基本都要复习过晚上12点才能躺下，第二天答题的时候倒是挺顺畅，一般是1个小时后交卷，有一科我就做了半个小时，交了，自己倒是没感觉太快，可着实吓着了同一考场的同学们。我出来是为了能早给枫打个电话问问，她那天下午正好是坐火车回家，我又没机会送她，打个电话还是应该的！这就是爱情的力量吧！

明天就能踏上回家的旅程了，不过可恶的是本来买的明早8：10的车，竟然那个点的车卖没了，给我拖到了12：20，这样我就必然得晚上才能回到家，现在想想，能够安安稳稳回去就满足了，晚点就晚点呗。

我怎么还是会关心

Jan 8th, 2012

本以为找到了一个靠谱的归宿，但任何事物都是在不断变化发展的，归宿这种非物质的东西也普遍遵循这条原理，如果你不好好经营，什么归宿都将变为泡影。

第一次由衷讨厌考试，它让我不能和枫好好打电话，好好发短信，让我不能正常休息（这纯粹是自己的原因）……

现在有时听到aki的消息还是会多少关注一下，看到枫的前BF也会稍稍看看，看到自己以前喜欢过的人也会多看两眼，我们的承诺虽然是永远只爱一个人，但爱的过程中并不是封闭的，有很多机会让我们遇到新的事物，那些意志不坚定的人容易在此时动摇。严格地讲我都算是吧——如果当初一直追aki，说不定也能有个不错的结果，如果……

“如果”的事情永远都不会发生在自己的身上，只是自己对于没有发生事情的期待让我们不断胡思乱想，想得多了说不定还真会让你去朝着那个方向做。

别考虑那些“如果”了，珍惜现在手中的，虽说给自己留条后路是大家普遍接受的做法，但真得不是每件事情都允许你给自己留条后路，还是有那么一些东西需要你全身心投入的，比如爱情：）

正常有感情的人遇到这些情况都会多关心一下，或是吃点小醋，这都是一个正常之人该有的表现，这样说的话，我还算正常：）

 

为了那些即将要走的人

Jan 8th, 2012

小狼和阿坤说要走了，不再继续上学了。

对于小狼要走我们原来宿舍的几个人已经感觉见怪不怪了，他说要走已经不是一次两次了，就和“狼来了”一样，我们很难再相信，不过又把每一次当作真的。

阿坤要走我是有些意外的。好像是考试前夕，他说都不上了还考什么试，我以为就是说着玩的，没想到今晚吃饭的时候证实，这也是真的。阿坤说要回家，要么和他爸爸干建筑材料，要么自己出去干点什么，在这里光是玩不假，而且专业不感兴趣，更促进了逃课和玩。只要是想好了我就不拦你，因为我坚信那句话——“如果你有目标，全世界都会给你让路”。

小狼为何要走我们现在也难以把握了，自己光说两年半在这里光是玩了，什么都没学到，再就是网恋了好几场，除了第一个从高中带过来的GF稍微靠谱点，剩下的都不靠谱，没计算过多久更新一个，反正我感觉这事长不了，到时候不光给自己带来巨大的痛苦，而且时间也搭进去了。你自己没有什么特别突出的特长，而且家里又不是富可敌国的环境，女孩为何要选择你？她们哪还会裸婚？！

你们都好好努力吧，在学业上的不努力最终都是要在别的地方补回来的！

这奖学金花得真快

Jan 7th, 2012

4号拿到了去年的所有奖学金，这里就不再说学校的各种不是了，白白拿了我们很多人的钱自己生利息去，最终不给我们分红！

其实从昨晚就开始看iPad2了，6号有apple store的优惠，不过无法派送到我家地址，感叹了些许，从网上看到有人拍下东西后打电话让客服修改了地址，顺利拿到货，只是去年11月实践的，我打算等白天考完试也这样做。谁成想不可能，客服不给修改地址，无法派送就是无法派送。

由于心中各种纠结，一时半会买不成，阿敏让我买些过年吃的零食，一买就是一下午，花了750+，晚上又买了个足疗机，花了680。终于在深夜，我买了iPad2 16GB WIFI版。

现在看看奖学金，只有1500块了，还能买什么呢？！

UPDATE：剩下的钱还了信用卡的欠款，然后买了HD518耳机，光了！

还让不让人支持正版啦？

Jan 3rd, 2012

这两天打算把喜欢的app买下来，但支付的过程中遇到一个巨大的问题，无法验证我的VISA卡。总提示安全码无效，无效你姐呀，就三位数字我怎么会看错？！而且其他信息都正确，包括有效期、卡号，而且我登陆了工行的网上银行检查信用卡设置，境外无卡支付、实时购汇等相关业务都开启了，你要说余额的话确实少点，人民币还剩下7块多，美元还0.08，我觉得不能是这样的问题吧？

又不想一下子充上50、100的放在那里不用，所以现在很郁闷！刚刚通过apple网站提交了一个支持，等他们发回邮件吧。

想支持下正版实在不太容易呀！

UPDATE：apple的kevin给我回复了邮件，称国内只有“某些银行”的信用卡能够使用，回复原文就是这样说的，似乎对这“某些银行”有些鄙夷的味道，看来我只能充值50、100那样用了，算啦，怎么都是支持正版，就这样吧！

UPDATE2：开了中行网银，在windows下装了itunes，然后充了50，买了第一个app。

被某些人坑了

Jan 3rd, 2012

元旦前的那个星期，从星期二开始考S2SH、WebService、ASP.Net，是企业方出题，还做出来一个在线考试系统，让我们在学校这边在线考试，第一年试点。老师为了不让企业来给我们监考，决定让我们在元旦前考试，这样就能争取到自己老师监考。枉我奋斗了3天，喝了六瓶咖啡（还是雀巢的丝滑好喝，比特浓的好喝多了）加一瓶增强型红牛（喝了就睡着了），考完试后被通知要重新笔试。

这一切都源于一个举报人，向院长举报老师协助我们作弊，而且10级的一个同学发答案的时候竟然发到了企业所在的那个群里，这下企业也找了院长，想不重考都难！总是我们班里的同学在群中大骂了好几天，誓要找出那个人是谁，还要“绳之以法”，说实在我也很讨厌这样，自己好不容易复习过去了，而且成绩也说的过去吧，但又所有的努力都作废，谁能淡定（除了举报人）？！

不过我们真的不该让这些本该正确的思想被埋没，本应是合法的要求，却被大多数人唾弃，我只能祝你。

2011，我的这一年（爱情篇）

Jan 1st, 2012

虽然现在还是一名大学生，不用想职场中的人们那样，在年末写总结，如果让我写的话我也绝对会认真书写，看看这一年我都做了什么，有什么成功的地方，有什么失败的地方，有什么值得骄傲的成绩，有什么辛酸苦楚……每一年看过来，总是由酸甜苦辣组成的，当这些味道都聚齐了之后，才能感到生活的丰满。

简单讲就是离开了枫，进入了aki的世界，又回归了枫。

三四月份的时候，枫说前男友想要和她重新开始，原本以为这样的话只能在电视剧中见到，没想到就真真切切地发生在我的身边，那时我好失望，真的很失望，我付出了这么多的努力，眼看就要看到曙光，她却这样向我泼了冷水，这盆水熄灭了我的激情，让当时的我真的放弃了她，从那以后再没有联系。

暑假里有机会和aki参加了数学建模比赛，虽然只有3天，但很累，收获也有，经历过就好了。从那以后我便开始喜欢aki了，现在来看只有3个月的时日，也算是一个不长不短的时间，期间都是我在主动追求她，自己也为此改变了不少，不过终究还是没能打动她，我自己不想再像以前那样浪漫也算是一个主要原因，aki对我来说太神秘了，有时表现的冷漠，有时又有点过于热情，谁受得了这样冷热交替。

在看《失恋33天》之前我给枫发了短信，问她是不是看过了，她说打算xxx时候看，我就疑惑了，你又不是单身看什么，她告诉我她又是单身了。那晚我意识到我的机会是不是又来了？不过我不想同时和两个人发展，这不是我的原则。后来到了平安夜的那个星期，我在焦急地等送给她们的礼物，可惜一样礼物都没来，我就只好跑到了枫的身边，算作是我对于她们两个之间做出了一个选择，现在来看，这个选择是正确的。

2011，我的这一年（事业篇）

Jan 1st, 2012

说得有点夸大，自己在学校里，主要的事业就是学习成绩了，今年没有拿到像去年那样那么大的奖，只拿到优秀奖参与奖这样的几张证书，不过由于去年的努力，拿到了国家励志奖学金，顺便也收下了学校的一等奖学金，终于有不少钱花了，本想买iPad2送给aki，但现在做什么都是徒劳的，因为有了枫。

面试了豆瓣的实习生，又一次发现了自己真得缺少很多知识。别人眼中的我有点神的味道，但真正去面试豆瓣这样的公司都会有问题，除了豆瓣的条件很高外，当然还有自己的能力远不够这种硬伤。光说没用，每次说说要是都不做等于自己打自己耳光！

后来偶然的机会给猛买网负责技术的blogkid发了一封邮件，有机会去实习了，不过学校这边不让走，说是到了大四都得继续看学校安排，实在受不了这样的环境，这也是我不愿意继续读研究生的一个主要原因，虽然我知道研究生基本可以不留在学校里，但那样的话基本也是被导师带着做项目，自己还得屁颠屁颠说学到了什么什么，想想真是不值得浪费青春在那里。
